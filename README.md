# EcommerceStore
# Northline Market - Basic E-Commerce Site
link: http://localhost:3000/

A basic full-stack e-commerce website built with HTML, CSS, JavaScript, Node.js, and a SQLite database.

## Features

- Product listing page
- Product search and category filtering
- Product details page
- Shopping cart with quantity controls
- User registration and login
- Authenticated checkout
- Order processing
- SQLite database for products, users, and orders
- Real product images from Unsplash

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js with an Express-style server
- Database: SQLite
- Authentication: Token-based login with hashed passwords
